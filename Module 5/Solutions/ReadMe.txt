When using the code solutions provided please ensure you match up the project path of the java classes with your own project path.

For installing the provided APKs, connect your Android phone to the PC and enable the transfer file mode. 
Open the device storage in the File explorer on the PC and copy the APK on the device (a suggestion is to copy it into the Downloads folder). 
You can now find the apk on the mobile phone, click on it and then "Install".
